# Personal Library Manager

## Overview
A detailed Python application that helps users organize books, track reading status, search their collection, monitor reading goals, and generate library statistics.

## Features
- Add books with title, author, genre, year, rating, status, and notes
- View full book collection
- Search by title, author, genre, or status
- Track read, unread, and currently reading books
- Generate library statistics
- Track yearly reading goal progress
- Delete books from the collection
- Store personal notes for each book

## Skills Demonstrated
- Python
- Functions
- Lists
- Dictionaries
- Loops
- Conditional Logic
- User Input
- Search Functionality
- Data Tracking
- Basic Analytics
- Menu-Based Program Design

## Why This Project Matters
This project demonstrates how Python can be used to organize personal information, search stored data, calculate progress, and generate useful summaries.
